# insta-explore
Exploring Instagram data collected with zeeschuimer

Jupyter Lab [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/jobreu/insta-explore/main)

RStudio [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/jobreu/insta-explore/main?urlpath=rstudio)


Demo notebook for importing and exploring Instagram data collected with the tool [zeeschuimer](https://github.com/digitalmethodsinitiative/zeeschuimer)

Created for the masters-level seminar "Erforschung von politischer Online-Kommunikation mit digitalen Methoden" ("Studying political online communication with digital methods") at the University of Duisburg-Essen (winter term 2025/2026).
